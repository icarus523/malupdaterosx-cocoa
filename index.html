malupdaterosx-cocoa by chikorita157

MAL Updater OS X GUI, rewritten in Cocoa. For more updates, check the MAL Client OS X Dev Log

MAL Updater OS X is a native, Cocoa scrobbler for Mac OS X. Not to be confused by TakakoShimizu's project that has the same name.

Unlike TakakoShimizu's version, MAL Updater OS X has an easy to set up GUI so new users don't have to get frustrated over Ruby scripts and such. The scrobbling is controlled by the Status Item Menu and can be monitored with a HUD Status Window that shows the status of the scrobbler and what was scrobbled last.

Like with TakakoShimizu's version, MAL Updater OS X Cocoa has Growl notifications built in to insure what the scrobbler is doing. In addition, MAL Updater OS X has Sparkle, which keeps the program up to date without launching your browser to get the update.

Dependencies

To Build:

BGHUDAppKit.framework
Sparkle.framework (GC Version)
OgreKit.framework
Growl.framework
JSON.framework

OS requirements

Mac OS X 10.5 or higher, PPC and Intel

Install

Unzip the MAL Updater OS X archive and run MAL Updater OS X.
